# Hyperlocal Marketplace

?? **Live Demo:** https://anushreer22.github.io/hyperlocal-marketplace

A local marketplace platform connecting vendors and customers in the same locality.

## ?? Features
- **Vendor Management** - Register, login, and manage products
- **Customer Shopping** - Browse products, add to cart, place orders
- **Admin Dashboard** - Manage vendors and oversee platform
- **Product Listings** - Categorized products with images

## ??? Tech Stack
- **Frontend:** HTML, CSS, JavaScript
- **Backend:** PHP, MySQL
- **Hosting:** GitHub Pages (Frontend), Render (Backend)

## ?? API Endpoints
- **Base URL:** https://hyperlocal-backend-3zcf.onrender.com
- **Vendor Auth:** /vendor/login_vendor.php, /vendor/register_vendor.php
- **Customer Auth:** /customer/login_customer.php, /customer/register_customer.php
- **Admin Auth:** /admin/login_admin.php
